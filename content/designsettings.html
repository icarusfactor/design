 Template Mailbox

 
Example Templates(Right-Click Copy Link): rcdt

Find More here: https://github.com/icarusfactor/designTmplPart/tree/main/template

 This mailbox is used to pull base HTML email pages to work off from and save them as drafts before final edit of them in Composer. You can move emails to this mailbox to create a new template and flag them to be recognized, no more than 10 templates can be recognized,if you have more than ten, then these will be ignored until you unflag other templates. If servers templates gets out of sync with browser editor it can be updated from here.Default templates to work with can be imported from here.

 This mailbox is used to pull HTML sections called Parts. You can move emails to this mailbox to create a new parts and flag them to be recognized, no more than 10 templates can be recognized,if you have more than ten, then these will be ignored until you unflag other templates. If servers parts gets out of sync with browser editor it can be updated from here. Default Part box to work with can be imported from here.
